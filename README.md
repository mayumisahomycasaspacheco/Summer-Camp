# Miauyummi's Personal Site
Hi! This is the repository for my personal website. I'm building it from scratch as part of my learning at this week-long camp.
## Sumer Camp - Day 1
- First, I started with just this, my index, where I placed a big title and a short description.
- After that, I added a nice image to make my website a bit more welcoming.
- Then, as you can see, I added some CSS styles, such as centering my page, giving it some color, and changing the typography.
- And to be able to present a more complete project, I added two sections: "About me" and "Things i love".
That's it, that was my work for day 1, I hope it can be understood and separated from day 2, thank you very much!
## Summer Camp - Day 2
- Now my website has its own link for each section.
- I also have a small top icon (favicon).
- And I learned how to use spacing.
- Surprise! We have a 404 error page section.
## Summer Camp - Day 3
- I transformed my simple link text into big, beautiful, and completely clickable buttons.
- I created a centralized container card layout to group my navigation links elegantly.
- I added cute background images to my main section and custom layout elements, handling margins perfectly to remove all those annoying blank spaces.
- I successfully implemented CSS Variables (`:root`) to manage my palette colors dynamically and fulfill all my requirements!
